# -
基于stc8A系列单片机和ov7670模块制作的人脸识别系统
实验现象：        下载程序后,打开上位机软件选择正确配置，点击连续接收图像，可看到接收到的图像。
                       
接线说明：         1：STC8A单片机-->LED&交通灯模块
                            P00-->D1       
                    2：STC8A单片机-->PZ-OV7670摄像头模块
                            GND-->GND
                                3.3V-->3.3V
                                P10-->SCL
                                P11-->SDA
                                P33-->WRST
                                P36-->RRST
                                P37-->OE
                                P20-P27-->D0-D7
                                P35-->RCLK
                                P34-->WEN
                                P32-->VSYNC
